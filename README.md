# Messy Theatre
<img width="960" height="540" alt="MessyTheatre01" src="https://github.com/user-attachments/assets/27772e6d-3e8c-4207-9149-2a2df4746e88" />

Messy Theatre is a third-person, multiplayer, casual ragdoll arena game created in three days for the Global Game Jam 2024.

Developed by a team of four under the theme "Make Me Laugh", players play in a PvP arena to appease an overseeing king, trying to make them laugh. Players can slap each other, pick up and throw each other, throw objects at each other, and deliberately get hit by arena hazards in an effort to make the king laugh.
